## 三、音乐术语（50+）

### 速度 Tempo

| 术语 | BPM 范围 | 感觉 |
|------|---------|------|
| Largo | 40-60 | 极慢，庄严的，如葬礼进行曲或深沉冥想 |
| Adagio | 60-70 | 缓慢的，从容优雅，如慢板抒情乐章 |
| Andante | 70-85 | 行走般的速度，自然舒适，如散步叙事 |
| Moderato | 85-100 | 中等速度，不急不缓，如日常对话 |
| Allegretto | 100-115 | 稍快的，轻松活泼，如愉快出行 |
| Allegro | 115-140 | 快速的，充满活力，如奔跑或欢庆 |
| Vivace | 140-160 | 活泼快速的，生机勃勃，如追逐或舞蹈 |
| Presto | 160-200 | 极快的，急促紧张，如紧急逃跑或高潮 |

> **Prompt Tips**: 可以直接写 BPM 数值(如 "120 BPM")或用术语(如 "allegro pace")，或用比喻(如 "walking pace" = andante)。

### 力度 Dynamics

**从弱到强**:
```
ppp (极弱) → pp (很弱) → p (弱) → mp (中弱) → mf (中强) → f (强) → ff (很强) → fff (极强)
```

**力度变化**:
| 术语 | 含义 | Prompt 用法 |
|------|------|------------|
| Crescendo | 渐强 | "gradually building intensity" |
| Diminuendo / Decrescendo | 渐弱 | "slowly fading away" |
| Sforzando (sfz) | 突强 | "sudden accent / sharp attack" |
| Subito piano | 突弱 | "sudden drop to quiet" |

| 术语 | 含义 | Prompt 描述方式 |
|------|------|----------------|

| Staccato | 断奏，音符短促分离 | "short, detached, bouncing" |
