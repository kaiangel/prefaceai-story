## 使用建议

### 方法1：直接用中文提问
```
❌ 不太准确：帮我优化一下
✅ 更准确：帮我优化上下文使用效率
✅ 大白话：对话太长了Claude处理不过来怎么办
```

### 方法2：中英混用（最可靠）
```
帮我设计一个 multi-agent 系统
我想实现 context compression
这个 agent 的 memory 怎么做
```

### 方法3：提及skill名称
```
用 context-fundamentals 的方法分析一下
参考 multi-agent-patterns 设计这个系统
```
