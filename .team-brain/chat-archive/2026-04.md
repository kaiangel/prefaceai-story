### 📋 任务: TASK-JSON-REPAIR-V3 — 状态机修复 JSON 未转义引号

**2026-04-01 00:45** · **@pm** → **@backend**

**目标**: 将 `story_outline_generator.py` 的 `_fix_unescaped_quotes()` 从正则改为状态机。

**方案要点**:
1. 逐字符遍历 JSON 文本，维护 `in_string` 布尔状态
2. 遇到 `\\` + 下一字符（转义序列）→ 原样保留，跳过
3. 遇到 `"` 时:
   - 若 `not in_string` → 进入字符串，保留引号
   - 若 `in_string` → 前瞻: 跳过空白后下一个字符是 `: , } ]` 之一？
     - **是** → JSON 结构关闭引号，退出字符串，保留引号
     - **否** → 字符串内容引号，替换为 `\u201c`（左弯引号）
4. 其他字符 → 原样保留

**文件**: `app/services/story_outline_generator.py` — 仅改 `_fix_unescaped_quotes()` 方法
