## 四、页面模块详解

### 4.1 模块总览

```
┌─────────────────────────────────────────────────────────┐
│  1. Header (固定顶部)                                    │
├─────────────────────────────────────────────────────────┤
│  2. Hero Section (全屏条漫展示)                          │
├─────────────────────────────────────────────────────────┤
│  3. Value Proposition (三大差异化优势)                   │
├─────────────────────────────────────────────────────────┤
│  4. Pipeline (FrameSpark™ 五阶段)                       │
├─────────────────────────────────────────────────────────┤
│  5. Showcase (作品展示画廊)                              │
├─────────────────────────────────────────────────────────┤
│  6. Stats (技术指标)                                     │
├─────────────────────────────────────────────────────────┤
│  7. CTA (邀请码申请)                                     │
├─────────────────────────────────────────────────────────┤
│  8. Footer                                               │
└─────────────────────────────────────────────────────────┘
```

### 4.2 各模块详细规格

#### 模块1: Header

**布局**: 固定顶部，高度 64px

**元素**:
- Logo（左）: `✦ 序话Story`
- 导航（中）: 功能 | 作品 | 技术 | 联系
- CTA（右）: `申请内测` 按钮

**行为**:
- 滚动时背景变为 `rgba(18, 18, 18, 0.98)` + 毛玻璃效果
- 移动端：汉堡菜单

#### 模块2: Hero Section

**布局**: 全屏高度，垂直居中内容

**元素**:
1. Badge: `🟢 内测申请开放中`
2. 主标题: `一句话，一个完整故事` （"完整故事"用品牌渐变色）
3. Slogan: `每个人都有自己的故事` （思源宋体，文学感）
4. CTA按钮: `开始你的故事` + `看看效果`
5. **条漫横向滚动展示**:
   - 4张条漫图片横向排列
   - 支持左右滑动
   - 故事切换器（两个圆点，切换都市亲情/赛博朋克）

**条漫图片路径**:
```
故事A（都市亲情）:
- test_output/comic_full_story_v2_20260127_opt005/with_text_images/shot_01.png
- test_output/comic_full_story_v2_20260127_opt005/with_text_images/shot_02.png
- test_output/comic_full_story_v2_20260127_opt005/with_text_images/shot_03.png
- test_output/comic_full_story_v2_20260127_opt005/with_text_images/shot_04.png

故事B（赛博朋克）:
- test_output/comic_full_story_v2_cyberpunk/with_text_images/shot_01.png
- test_output/comic_full_story_v2_cyberpunk/with_text_images/shot_02.png
- test_output/comic_full_story_v2_cyberpunk/with_text_images/shot_03.png
- test_output/comic_full_story_v2_cyberpunk/with_text_images/shot_04.png
```

**动效**:
- 条漫图片 hover 时轻微放大 (scale 1.02)
- 故事切换时图片淡入淡出 (700ms)

#### 模块3: Value Proposition

**布局**: 三列卡片（移动端单列）

**内容**:
| 图标 | 标题 | 描述 |
|------|------|------|
| ⚡ | 即出即发 | 生成即可发布的条漫或短视频，无需后期处理，直接上传社交平台 |
| 👥 | 角色一致性 | 独创混合模型架构，3人场景100%一致，6人场景90%+一致 |
| 🎬 | 双重输出 | 条漫和短视频双形态输出，一次创作，多平台分发 |

**样式**:
- 背景: `--bg-secondary` (#1E1E1E)
- 卡片: `--bg-tertiary` (#2A2A2A)
- 卡片 hover: 上移 4px + 阴影

#### 模块4: Pipeline (FrameSpark™)

**布局**:
- 顶部: 品牌标识 `✦ FrameSpark™`
- 标题: `五阶段故事引擎`
- 内容: 5个阶段横向排列（移动端纵向）

**5个阶段**:
| # | 标题 | 描述 |
|---|------|------|
| 1 | 故事大纲 | AI理解你的创意，生成角色和情节框架 |
| 2 | 角色设计 | 生成详细的角色外貌、服装、性格设定 |
| 3 | 分镜脚本 | 专业镜头语言：景别、角度、构图全自动 |
| 4 | 画面生成 | 高质量图像生成，角色100%一致性保证 |
| 5 | 成品输出 | 条漫/短视频双形态，即刻可发布 |

**样式**:
- 阶段卡片左边框: 3px 品牌色
- 阶段编号: 圆形品牌色背景

#### 模块5: Showcase

**布局**:
- 标题: `用户故事展示`
- Tab切换: 全部 | 都市情感 | 科幻冒险 | 古风武侠
- 图片网格: 4列（移动端2列）

**图片路径**: 使用已有的测试输出图片
```
都市亲情: test_output/comic_full_story_v2_20260127_opt005/with_text_images/shot_05-10.png
赛博朋克: test_output/comic_full_story_v2_cyberpunk/with_text_images/shot_01-08.png
```

**交互**:
- hover: 图片放大 (scale 1.05) + 显示故事标题 overlay

#### 模块6: Stats

**布局**: 4列数据展示（移动端2列）

**数据**:
| 数值 | 标签 |
|------|------|
| 100% | 3人场景角色一致性 |
| 90%+ | 6人场景角色一致性 |
| 80+ | 支持视觉风格 |
| ≤80ms | 音画对齐精度 |

**样式**:
- 数值: 大号字体 + 品牌渐变色
- 上下边框: 1px 分隔线

#### 模块7: CTA

**布局**: 居中，最大宽度 600px

**元素**:
1. 标题: `开始创作你的故事`
2. 副标题: `申请内测资格，成为第一批体验者`
3. 表单: 邮箱输入框 + `申请内测` 按钮
4. 说明: `我们会在 24 小时内审核并发送邀请码`

**背景**: 品牌色渐变叠加 (10% opacity)

**交互**:
- 表单提交后显示成功提示
- CTA按钮有发光动效 (glow animation)

#### 模块8: Footer

**布局**: 4列（品牌 + 产品 + 支持 + 关于）

**内容**: 参见架构文档

## 五、视觉规范速查

### 5.1 配色

```css
/* 品牌色 - 暖琥珀 */
--brand-primary: #FF9500;
--brand-gradient: linear-gradient(135deg, #FF9500 0%, #FFD93D 100%);

/* 背景色 - 故事感深色 */
--bg-primary: #121212;
--bg-secondary: #1E1E1E;
--bg-tertiary: #2A2A2A;

/* 文字色 */
--text-primary: #FFFFFF;
--text-secondary: #E0E0E0;
--text-tertiary: #9E9E9E;
```

### 5.2 字体

| 用途 | 字体 |
|------|------|
| Hero标题 | 思源黑体 800 |
| Slogan | 思源宋体 |
| 正文 | 思源黑体 400 |
| 英文/数字 | Inter |

| 场景 | 时长 |
|------|------|
| hover状态 | 200ms |
| 内容展开 | 350ms |
| 故事切换 | 700ms |
| CTA发光 | 2000ms (循环) |
