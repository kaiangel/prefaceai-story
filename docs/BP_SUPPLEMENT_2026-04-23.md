## 1. 竞品差异化（视频生成 + AI 漫剧 / 条漫赛道）

**关键事件（2026-04-26）**：OpenAI Sora 关停。日烧约 $100 万算力、App 终身总收入仅约 $210 万、MAU 从峰值 100 万跌至 50 万；Disney $10 亿合作告吹。**这反向证明"纯视频片段生成"的 C 端商业模型不成立**——用户买的不是片段，是有结构、有角色、可分享的完整内容。序话Story 的"故事 Pipeline + 完整作品输出"路径与之相反。

### 真威胁排序

1. **百度文心魔法漫画**（2025-11 上线）— 产品定位最接近，C 端一句话条漫；缺 BGM、缺动态、靠参考图（非系统级一致性）
2. **字节即梦**（Seedance 2.0，2026-02）— 多镜头故事板+12 张参考图，但仍是"工具积木"非"完整产品"，且字节内部优先级低（CapCut/抖音/豆包/Coze 互相挤资源）
3. **MiniMax 海螺 + 快看 AI 互动漫画**— 角色一致性技术全球领先，但条漫专项产品弱

### 序话最难被复制（按强度排序）

1. **FrameSpark™ Pipeline 工程护城河**：5 阶段编排（故事/角色/剧本/分镜/图像）+ 跨帧一致性的 prompt 工程积累 — 大厂有模型没工程细节，初创有积极性没资源，序话在甜蜜点（窗口期估计 6-12 个月）
2. **BGM × 条漫情绪密度**：竞品都把 BGM 视为独立模块，序话做的是原生融合 — 同样故事，有 BGM 版本社交传播率数倍
3. **品类认知**：在用户心智里"序话 = 有感觉的 AI 条漫"vs"竞品 = 工具"

> 注：Founder 原列竞品"注先 / 秒东"经多源调研推断分别最可能对应**即梦（字节）**和**一帧秒创（aigc.yizhentv.com，200 万+创作者）**。若理解有误请告知。

| TTS | Doubao 火山引擎 | ~$0.001 |
| **合计** | | **~$1.85 / 条 ≈ ¥13.5** |

### 三条路径主次

**主线（M0-M18）：订阅 C 端**
- Free 3 条/月（带水印）/ Lite ¥49（2 条 + ¥25/超出）/ Pro ¥199（10 条 + ¥20/超出）/ Pro Max ¥499（25 条 + ¥18/超出）
- 当前成本结构下毛利率 **30%**（图像 API 占 72% 成本，是天花板）
- Pre-A 前完成自建推理可至 60%
- **关键约束**：CAC 必须靠有机增长压在 ¥500 内 — 保守 LTV/CAC=0.19（不健康）/ 基础 2.2 / 乐观 19.4
- → 天使期不能依赖付费投放，必须靠口碑 + UGC + KOL

**次线（M6+）：序话Story API**
- vs Runway / Vidu / 可灵 API 的差异化：**一站式 Pipeline API**（一接口完成 LLM + 图像 + BGM + TTS 全流程），不是单模型包装
- 套餐：Developer ¥199 / Startup ¥999 / Growth ¥4999 / Enterprise 协商
- M12 目标：20-30 客户，MRR ¥30K
- 价值：开发者生态 + 渠道放大效应

**延展（Pre-A 之后才规模化）：B2B 企业服务**
- 客户：IP 方 / 出版社 / MCN / 品牌方 / 教育机构 / 游戏公司
- 客单价：¥3-50 万/年
- 天使期内只接 **2-3 个标杆客户**（建议教育 + MCN），定价对折换深度合作 + 公开案例
- 毛利率目标 40-60%（含人力）

**不做（天使期）：视频版按量 C 端订阅**
- 视频版单条 $5-10，当前定价无法实现正毛利，需等视频 API 价格再降 50% 或自建
- 短视频是 Q2 MVP 验证、Pre-A 后才规模化

## 5. 让 BP "不自孩"的 5 条关键事实

1. **Sora 关停（2026-04-26）= 反向背书**

2. **NB2 = 商业化拐点（2026-02-26 商用）**

## 6. 单位经济与成本工程（2026-04-25 新增）

### Baseline（已验证）

单条故事生成成本 **$1.85**（图像 NB2 $1.34 / 72% + 文本 Sonnet 4.6 $0.50 / 27% + BGM Mureka $0.05 / 3% + TTS Doubao $0.001 / <0.1%）。Pro ¥199/月（10 条配额）当前毛利率 **30%**。

### 这不是问题，是路线图

| 月份 | 应用杠杆 | 单条成本 | Pro 毛利率 |
|------|---------|---------|-----------|
| M0 现状 | 串行 + 满 quota 订阅 | $1.85 | **30%** |
| **M3** | **+ 工程并行化（13.5 min → ~4.5 min UX 升级）+ Credits 制（实际消耗 70%）** | $1.85 × 70% | **53%** |
| M6 | + 产品减量（Stage D 6 张预览，平均 14 张） | $1.30 × 70% | **62%** |
| M9 | + Google API 量产议价（量到 100K 张/月） | $0.95 × 70% | **70%** |
| M12 | + Batch 部分场景（B 端 / API / 章节预生成） | $0.80 × 70% | **76%** |
| M18 | + 自建 SD 集群（部分流量） | $0.50 × 70% | **85%** |

### 4 个杠杆的可控性

1. **工程并行化（M1，完全可控）**：现有 `image_generator.generate_batch()` 已实现 `asyncio.Semaphore + asyncio.gather()`，pipeline 主路径只需替换 for 循环。耗时 13.5 min → ~4.5 min（达到 Midjourney Fast Mode 同等体验），不需要任何外部依赖。
2. **Credits 制定价（M2-M3，完全可控）**：Pro ¥199 = 200 Credits（1 张 = 1 Credit），未用 Credits 30 天结转。行业证据（Runway / Midjourney / Pika）：用户实际消耗 = 配额 60-70%。
3. **Google API 议价（M6+，部分可控）**：量到 100K 张/月可谈 30-40% 折扣。前置条件：Q3 月付费 1500+ 用户。
4. **自建推理（M18+，Pre-A 后规模部署）**：H100 集群 + Stable Diffusion 3 + 角色 LoRA，单图 $0.005-0.015（vs NB2 $0.067，降 80-90%）。前置 PoC 需验证多角色一致性可达 NB2 水平 85%+（M9 启动 PoC，CTO Ben + 1 名 ML 候选人）。

### 风险与反脆弱性

- **不依赖任何"我们等"的事**：M3 53% 毛利不需要 NB2 价降、不需要 Google 修 Batch Bug
- **不依赖单一杠杆**：4 个杠杆任何 1-2 个失效，剩余仍能推动毛利率上升
- **同期对标**：Midjourney 60% / Runway 50-55% / Pika 30-40% / Jasper 50-60%。**M9 我们达 70%，已超主流 AI Wrapper**；M18 达 85%，接近纯 SaaS 行业基准（Notion 90% / Salesforce 80%）
