## 参考案例概述

**故事线**: 情侣逛街 → 男友拍照不好 → 女主生气不说话 → 男友道歉 → 女主回忆前任冷暴力 → 领悟沟通重要性 → 主动道歉和解

**角色设定**:
| 角色 | 外貌特征 | 服装特征 |
|------|----------|----------|
| 女主 | 红棕色波浪长发、脸颊有痣、柳叶眉 | 浅蓝开衫、珍珠项链/蝴蝶结颈饰 |
| 男主 | 黑色短发微乱、剑眉 | 深蓝夹克、灰色内搭、牛仔裤 |
| 前任 | 深棕色长发、耳钉、冷峻眉眼 | 深色衬衫、红色系打光 |

## 1. 文字内嵌效果

### 1.1 对话气泡

**参考图**: 804, 808, 812, 813, 814, 815

| 验收点 | 标准 | 权重 |
|--------|------|------|
| 气泡形状 | 白色圆角矩形，带尖角指向说话者 | 必须 |
| 文字清晰度 | 中文文字100%可读，无模糊/变形 | 必须 |
| 文字颜色 | 黑色或深灰色（对比度≥4.5:1） | 必须 |
| 气泡位置 | 靠近说话者头部，不遮挡面部表情 | 重要 |
| 气泡尖角 | 指向正确的说话者 | 重要 |
| 字体风格 | 与条漫风格协调（非系统字体感） | 加分 |

**验收清单**:
- [ ] 气泡是否为白色圆角矩形
- [ ] 文字是否100%可读
- [ ] 气泡尖角是否指向说话者
- [ ] 气泡是否遮挡了关键内容

---

### 1.2 心理旁白（黑底白字）

**参考图**: 804顶部, 805底部, 806顶部+底部, 807底部, 809底部

| 验收点 | 标准 | 权重 |
|--------|------|------|
| 背景色 | 纯黑或深灰（透明度≥90%） | 必须 |
| 文字颜色 | 纯白 | 必须 |
| 文字清晰度 | 100%可读 | 必须 |
| 位置 | 顶部或底部，不遮挡主要人物 | 必须 |
| 边缘处理 | 与画面自然衔接，无硬切边 | 重要 |
| 行距 | 舒适可读（约1.5倍行高） | 重要 |

**验收清单**:
- [ ] 黑色区域是否足够深（≥90%不透明度）
- [ ] 白色文字是否清晰可读
- [ ] 文字区域是否遮挡了关键画面
- [ ] 多行文字行距是否合适

---

### 1.3 叙事旁白（白底黑字）

**参考图**: 817顶部, 818中部

| 验收点 | 标准 | 权重 |
|--------|------|------|
| 背景色 | 白色或浅色（可带浅边框） | 必须 |
| 文字颜色 | 黑色或深色 | 必须 |
| 文字清晰度 | 100%可读 | 必须 |
| 与画面融合 | 不突兀，与整体风格协调 | 重要 |
| 边框处理 | 可有细边框增加层次感 | 加分 |

**验收清单**:
- [ ] 白色/浅色背景是否清晰
- [ ] 黑色文字是否可读
- [ ] 文字框与画面是否协调

## 2. 合成效果

### 2.1 分屏对比效果

**参考图**: 811

| 验收点 | 标准 | 权重 |
|--------|------|------|
| 分割线 | 垂直分割，位置居中或三分法 | 必须 |
| 两侧内容 | 形成明显对比（人物/场景/色调） | 必须 |
| 分割清晰度 | 分割线清晰可见或隐式区分明显 | 必须 |
| 两侧风格统一 | 虽有对比但属于同一画面 | 重要 |
| 叙事逻辑 | 对比有意义（如现任vs前任） | 重要 |

**验收清单**:
- [ ] 分屏是否清晰可辨
- [ ] 两侧内容是否形成有意义的对比
- [ ] 分屏效果是否自然融入画面
- [ ] 左右两侧风格是否统一

---

### 2.2 回忆碎片效果

**参考图**: 809

| 验收点 | 标准 | 权重 |
|--------|------|------|
| 碎片形状 | 不规则多边形或裂痕状 | 必须 |
| 碎片内容 | 包含回忆场景/人物的片段 | 必须 |
| 色调处理 | 碎片内容色调偏暗/灰/去饱和 | 重要 |
| 位置分布 | 环绕主角或发散分布 | 重要 |
| 边缘处理 | 碎片边缘有锐利感或光晕效果 | 加分 |
| 主体突出 | 主角仍是视觉焦点 | 必须 |

**验收清单**:
- [ ] 是否有明显的碎片化回忆效果
- [ ] 碎片内容是否与回忆相关
- [ ] 碎片色调是否与主画面有区分
- [ ] 主角是否仍是视觉焦点

### 2.3 画中画效果（手机屏幕）

**参考图**: 815, 816

| 验收点 | 标准 | 权重 |
|--------|------|------|
| 手机外形 | 清晰的手机轮廓（边框、摄像头等） | 必须 |
| 屏幕内容 | 屏幕内图像清晰可辨 | 必须 |
| 嵌入自然度 | 手机在画面中位置/角度自然 | 必须 |
| 比例正确 | 手机与手/人物比例协调 | 重要 |
| 屏幕风格 | 屏幕内画面与条漫风格统一 | 重要 |
| 反光/细节 | 可有轻微屏幕反光增加真实感 | 加分 |

**验收清单**:
- [ ] 手机外形是否清晰
- [ ] 屏幕内容是否可辨识
- [ ] 手机在画面中是否自然
- [ ] 手机比例是否正确

## 3. 表情细腻度

### 3.1 表情评判标准

| 情绪类型 | 关键特征 | 参考图 |
|----------|----------|--------|
| **不悦/生气** | 眉头微皱、嘴角下撇、眼神不看对方 | 804 |
| **委屈** | 眉头微皱、眼眶泛红感、嘴唇轻抿 | 805女主 |
| **困惑** | 眉毛上挑、眼睛睁大、嘴微张 | 805男主 |
| **若有所思** | 眼神望向远方、表情平静带忧郁 | 806 |
| **道歉/恳求** | 眉头紧皱、双手合十、汗滴符号 | 808 |
| **震惊/恍惚** | 眼睛睁大、瞳孔放大、嘴微张 | 809 |
| **冷漠** | 眼神空洞、嘴角平直、面无表情 | 810 |
| **释然/解释** | 眉毛舒展、眼神柔和、嘴角微扬 | 812, 813 |
| **开心/宽慰** | 眼睛弯成月牙、嘴角上扬、脸颊微红 | 814, 815 |

### 3.2 表情验收清单

- [ ] **不悦**: 眉头是否微皱？嘴角是否下撇？
- [ ] **委屈**: 是否传达出"想说又不想说"的复杂情绪？
- [ ] **困惑**: 眉毛是否上挑？眼神是否迷茫？
- [ ] **道歉**: 是否有恳切感？肢体语言是否配合？
- [ ] **震惊**: 眼睛是否睁大？表情是否有"恍然大悟"感？
- [ ] **冷漠**: 是否传达出"不在乎"的疏离感？
- [ ] **释然**: 表情是否从紧张变为放松？
- [ ] **开心**: 笑容是否自然？是否传达出真诚的快乐？

## 4. 风格一致性

### 4.1 画风验收标准

| 维度 | 标准 | 权重 |
|------|------|------|
| 线条风格 | 15张图线条粗细、清晰度一致 | 必须 |
| 色彩风格 | 整体色调统一（柔和、偏暖或偏冷） | 必须 |
| 人物比例 | 头身比、五官比例跨图一致 | 必须 |
| 背景处理 | 虚化程度、细节层次统一 | 重要 |
| 光影风格 | 阴影、高光处理方式一致 | 重要 |
| 整体氛围 | 韩漫/国漫条漫风格贯穿始终 | 必须 |

### 4.2 风格漂移检测清单

- [ ] 是否有某张图突然变成写实/卡通/其他风格？
- [ ] 是否有某张图线条突然变粗/变细？
- [ ] 是否有某张图色调突然变暗/变亮？
- [ ] 是否有某张图人物比例明显不同？
- [ ] 整体15张图放在一起是否像"同一个故事"？

## 5. 角色一致性

### 5.1 角色辨识度验收标准

| 角色 | 必须保持的特征 | 可接受的变化 |
|------|----------------|--------------|
| **女主** | 红棕色波浪长发、脸型、五官比例 | 服装细节、发型微调、妆容浓淡 |
| **男主** | 黑色短发、脸型、眉形 | 表情变化、服装细节 |
| **前任** | 深色长发、耳钉、冷峻气质 | 出场时服装、光影色调 |

### 5.2 角色一致性检测清单

**女主 (15张中需保持一致)**:
- [ ] 发色是否始终为红棕色？
- [ ] 发型是否始终为波浪长发？
- [ ] 脸型是否跨图一致？
- [ ] 珍珠项链/颈饰是否出现？
- [ ] 是否在所有图中一眼可认出是同一人？

**男主 (出现的图中需保持一致)**:
- [ ] 发色是否始终为黑色？
- [ ] 发型是否始终为短发微乱？
- [ ] 蓝色夹克是否一致？
- [ ] 是否在所有图中一眼可认出是同一人？

**前任 (回忆场景中)**:
- [ ] 是否与男主明显不同？
- [ ] 是否有冷峻/疏离的气质？
- [ ] 深色系服装/耳钉等特征是否体现？

## 6. 评分标准

### 6.1 单项评分（1-5分）

| 分数 | 评价 | 标准 |
|------|------|------|
| 5 | 优秀 | 完全达到参考案例水平，可直接发布 |
| 4 | 良好 | 接近参考案例，有小瑕疵但不影响阅读 |
| 3 | 及格 | 基本可用，有明显但可接受的问题 |
| 2 | 不及格 | 问题明显，影响阅读体验 |
| 1 | 严重不合格 | 完全不可用 |

### 6.2 各维度权重

| 维度 | 权重 | MVP及格线 |
|------|------|-----------|
| 文字内嵌效果 | 25% | ≥3分 |
| 合成效果 | 20% | ≥3分 |
| 表情细腻度 | 20% | ≥3分 |
| 风格一致性 | 20% | ≥4分 |
| 角色一致性 | 15% | ≥4分 |

### 6.3 综合评分计算

```
综合分 = 文字内嵌×0.25 + 合成效果×0.20 + 表情细腻度×0.20 + 风格一致性×0.20 + 角色一致性×0.15
```

**MVP通过标准**: 综合分 ≥ 3.5 且所有单项 ≥ 3分
